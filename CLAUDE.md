## Conta Azul (MCP)
Você tem acesso direto à API do Conta Azul via servidor MCP local.
Use SEMPRE as ferramentas abaixo — nunca peça token ou credenciais.

Ferramentas disponíveis:
- `listar_clientes` — lista e filtra clientes por nome, CNPJ, e-mail
- `buscar_cliente` — detalhes de um cliente pelo ID
- `listar_cobrancas` — cobranças por status/período/cliente
- `listar_contas_receber` — contas a receber
- `listar_notas_fiscais` — notas fiscais emitidas
- `resumo_financeiro` — visão consolidada financeira
